A MediaWiki extension ([http://www.mediawiki.org/wiki/Extension:SuppleRedirect Extension:SuppleRedirect]) provides supplementary redirect from empty article to other site e.g. divide a site.

Please see for details:
* https://kimagurenote.net/kn/SuppleRedirect (Japanese)
* https://www.mediawiki.org/wiki/Extension:SuppleRedirect (English)
